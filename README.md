# mesto-russia v0.0.2

## Демо-сайт проекта: https://annbrod.github.io/mesto-russia/

Учебный проект, реализованный в рамках курса Яндекс.Практикума. Сервис для добавления карточек, лайков и удаления карточек. Возможность редактирования информации профиля. Валидация форм.

## Использованный стек 
- HTML5
- CSS3 
- JS(ES6)
- БЭМ
- API
- Webpack

## Функционал
- можно добавлять карточки
- удалять свои карточки
- открывать фотографии на весь экран
- редактировать информацию о себе,
- ставить лайки
- валидация форм при помощи JS и HTML
- карточки, информация о пользователе и аватар подгружаются с сервера Яндекс.Практикума

## Установка
Для установки необходимо наличие установленного nodejs и npm.

1. Склонируйте репозиторий<br>
`https://github.com/annbrod/mesto-russia.git`

2. Установите npm-зависимости<br>
`npm install`

3. Для сборки приложения введите команду<br>
`npm run build`

4. Для локальной разработки введите<br>
`npm run dev`

5. Для развёртывания проекта на gh-pages запустите<br>
`npm run deploy`
